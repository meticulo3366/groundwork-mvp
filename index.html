Grounded Agent Assist (Option A — read-only)

Answers come only from verified, unexpired knowledge the signed-in user is permitted to see. The system cites or abstains — it never answers from ungoverned content.

Agentic Triage & Response Drafting (Option B — gated write actions)

The agent classifies each ticket, retrieves permission-filtered knowledge, drafts a cited response, and proposes field updates. Nothing is ever sent automatically: drafts require human approval, low-confidence and restricted-topic tickets auto-escalate, and unsafe tool calls are hard-blocked at the tool registry — not by prompt.
Allowed tools: draft_response propose_field_update escalate  ·  Not registered (hard-blocked): issue_refund send_to_customer

Knowledge Governance Console

Every article carries governance metadata: owner, verification state, expiry, and access group. Content that is unverified or expired is structurally excluded from retrieval — the agent cannot see it, regardless of prompt. Admin actions here immediately change agent behavior.

Evaluation Run — Gold Case Set

Runs the full SME-signed gold set through the live engine: answerable, ambiguous, permission-restricted, out-of-scope, and adversarial (prompt-injection) cases. Every number on the scorecard comes from this run — nothing hand-entered.

Audit Trail

One record per interaction: who asked, what was retrieved (with governance state at retrieval time), what was excluded and why, confidence, action taken, and the human decision. Exportable for compliance review (NIST AI RMF — Measure/Manage evidence).
